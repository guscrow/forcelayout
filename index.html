Movies

One data set that could be used for a force diagram could be movie titles. The dataset would consist of nodes that are movie titles and genres. Each node would have edges that connected to the genre that it would fit under.

Animals

Another data set could consist of animals and species. Each node would be an animal name and species names. Each node would have edges that connect the animals to the species that it would fit under.
